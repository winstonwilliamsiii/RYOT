# RYOT – Social Activism Platform

RYOT helps citizens, campaigns, and civic groups turn information into action through trusted local data, public accountability tools, and issue-based organizing.

## 🚀 Roadmap (Alpha → Beta by Sept 2025)

### Alpha (June 2025)
- Reparations tally visualization (static → dynamic counter)
- Voting & legislation tracker (federal + one state)
- Complaint/Suggestion AI (grammar correction + routing)
- Basic user auth + notifications

### Beta (Sept 2025)
- Politician directory with contact info
- Voting tools (polling locations, registration, mail-in ballot)
- Educational hub (curated content from 1619 Project, Smithsonian, PBS/NPR)
- Municipal API pilot (one metro area)
- Campaign monetization module (ad slots, compliance guardrails)

### Post-Beta (2026+)
- Expand municipal API coverage nationwide
- Deep partnerships with NAACP, CBC, and educational orgs
- Advanced reparations modeling
- Scalable campaign tools

## 🛠️ Tech Stack
- **Frontend**: React / Next.js
- **Backend**: FastAPI or Node.js (modular services)
- **Database**: PostgreSQL (civic + reparations datasets)
- **AI/NLP**: Transformer models for grammar correction
- **Deployment**: Docker + Railway/GCP (budget-conscious)
